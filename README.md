# 中国传统文化及故事知识库

这是一个按照省份和地区分类的中国传统文化知识库。

## 内容范围

- 民间故事
- 神话传说
- 非物质文化遗产
- 地方戏曲
- 节庆习俗
- 饮食文化
- 传统工艺
- 历史人物
- 地方掌故

## 目录说明

- data/provinces/：按省份整理的主知识库
- data/regions/：按城市、地区整理的补充资料
- sources/videos/：视频来源记录
- sources/books/：书籍来源记录
- sources/websites/：网页来源记录
- templates/：条目模板
- indexes/：索引文件
- notes/：临时笔记
