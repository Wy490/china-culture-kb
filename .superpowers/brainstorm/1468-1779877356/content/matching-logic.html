Claude Code 分析维度
主题识别：故事核心主题（孝道、忠义、爱情、因果报应等）
地理线索：文中提及的地名、省份、河流、山脉 → 优先检索对应省份
人物识别：故事中的关键人物名 → 检索历史人物条目
类型判断：神话传说？民间故事？节庆习俗？ → 同类型优先匹配
相似度评级标准
高度匹配：同一故事/同一人物/同一地点的核心叙事基本一致
中度匹配：主题相同、有共同人物或地点，但叙事差异较大
低度匹配：仅类型或主题相关，具体内容差异显著
kb_supplement 补充逻辑
三维度补充
维度1：版本差异
同一故事在不同省份的版本
如"白蛇传"在江苏、浙江、安徽的叙事差异
→ kb_search 同关键词跨省检索
维度2：同地同类
同一省份同一类型的其他条目
如江苏的民间故事群、四川的地方戏曲
→ kb_search 按省份+类型检索
维度3：关联网络
关联人物、地点、事件
如梁山伯 → 相关祝英台 → 万松书院
→ kb_search 关键词交叉检索
补充结果格式
### 补充内容
**版本差异**：[[浙江]]版本中白蛇修炼千年而非五百年
**同地同类**：江苏另有"柳毅传书"民间故事，同属爱情传说类
**关联网络**：相关人物—许仙；相关地点—西湖断桥、金山寺
