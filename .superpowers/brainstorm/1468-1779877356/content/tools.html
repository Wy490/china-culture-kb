MCP Server 工具定义

7个MVP工具的输入输出与行为

kb_fetch_video

从B站视频抓取内容
输入
bv_id (BV号) 或 url (视频链接)
输出
标题、UP主、发布时间、简介、字幕/弹幕摘要、视频中提及的故事名称列表
实现
调用B站API获取视频信息；yt-dlp提取字幕；AI提取故事名称

kb_fetch_article

从网页文章抓取内容
输入
url (文章链接)
输出
标题、作者、发布时间、正文(Markdown)、来源平台
实现
WebFetch抓取HTML → 转Markdown；记录来源元数据

kb_verify_source

核查来源可信度
输入
source_type, source_url, source_author, claims (待核实内容)
输出
可信度评级(可靠/基本可靠/待核实/存疑)、核查结果、待核实点清单
核查规则
官方来源→可靠；知名学者/出版社→基本可靠；自媒体→待核实；无来源→存疑

kb_add_entry

写入条目到省份文件
输入
完整条目字段（名称、省份、地区、类型、简介、故事梗概、文化意义、相关地点、关键词、来源、可信度、待核实点）
输出
写入确认、文件路径；跨省份条目自动在多个文件创建链接
行为
验证字段完整性 → 确定省份文件 → 按模板格式写入 → 更新索引

kb_match

语义匹配知识库条目
输入
story_text (用户上传的故事或观点)
输出
匹配条目列表，每个含：条目名称、匹配原因、相似度、省份、链接
实现
读取所有省份文件 → Claude语义对比 → 返回相关条目及匹配分析

kb_search

按条件检索知识库
输入
keywords, type (可选), province (可选), region (可选)
输出
匹配条目列表，含完整字段摘要
实现
遍历指定省份文件 → 关键词文本搜索 → 返回结果

kb_supplement

补充相关知识库内容
输入
entry_name 或 story_text
输出
相关条目：同一故事的不同地区版本、同类型条目、同地区条目、关联人物/地点
实现
定位当前条目 → 搜索同关键词/同省份/同类型 → 返回补充材料
