# 中国传统文化及故事知识库规则

本项目用于整理中国传统文化、民间故事、神话传说、非遗、地方戏曲、节庆、饮食、工艺、历史人物、地方掌故与名胜古迹。

## 分类方式

优先按照省级地区分类，文件位于：

data/provinces/

每个条目应归入对应省份文件。如果一个故事跨多个地区，可以在多个省份文件中都保留简要条目，并互相链接。

## 地理标注规范

### 地区字段格式

地区字段必须标注到**市/县**级别，格式为：`省→市→县/区`

示例：
- `湖南→永州→道县`（不写"永州道县"，用箭头分隔层级）
- `湖南→长沙→浏阳`（浏阳是县级市，归入长沙市）
- `湖南→岳阳→汨罗`（汨罗是县级市，归入岳阳市）
- `湖南→娄底→双峰`（双峰县归入娄底市，不是湘潭市）
- `湖南→湘西→凤凰`（自治州用州名，不写"湘西州凤凰县"，简写即可）

### 古今地名对照

当古代地名与现代地名不同时，必须在条目中标注对照关系：

格式：`古地名（今现代地名）`

示例：
- 楚国汨罗（今岳阳汨罗市）
- 长沙国（今长沙市及周边，西汉时范围远大于今长沙市）
- 长沙郡（今长沙市及周边，东汉郡治范围大于今市）
- 长沙府湘乡（今娄底双峰县，1951年从湘乡析出归娄底）
- 潭州（今长沙市，北宋时称潭州）
- 洪州分宁县（今江西修水县）
- 大庸/永顺府桑植（今张家界桑植县）

**关键原则**：条目归入**现代行政区划**对应的省份文件，但故事中涉及的古代地名必须标注现代对应。

### 籍贯与任职地——双轨收录

**在本省籍贯的名人**：地区字段标注籍贯/出生地。

**在外省籍贯但在本省任职/活动的名人**：也须在本省文件中收录，地区字段采用双轨格式：

`籍贯地（籍贯），关联地/任职关系`

示例：
- 辛弃疾：`山东→济南（籍贯），任湖南转运副使（南宋潭州）`
- 杜甫：`河南→巩义（籍贯），晚年漂泊湖湘（唐代长沙/岳阳）`
- 贾谊：`河南→洛阳（籍贯），贬长沙王太傅（西汉长沙国）`
- 张仲景：`河南→南阳（籍贯），任长沙太守（东汉长沙郡）`
- 陶侃：`江西→九江（籍贯），封长沙公（东晋长沙，食邑长沙）`

**任职名人收录模式**：

| 情况 | 收录方式 |
|------|----------|
| 名人在该地有丰富故事（≥3个独立故事情节） | → 独立历史人物条目 |
| 名人与该地关联仅为一个具体事件 | → 写入相关名胜古迹/地方掌故条目的"名人与它的故事"部分，不单独建条目 |

**互引规则**：已有独立人物条目的任职名人，在其他条目中只写简述+链接（如"杜甫晚年登岳阳楼写下《登岳阳楼》→详见[[杜甫——漂泊湖湘的晚年诗圣]]"），不重复人物完整故事。避免跨省重复——任职名人条目末尾标注"籍贯详见→[[XX省相关条目]]"。

## 条目字段

每个文化条目必须包含：

- 名称
- 省份
- 地区（按上述地理标注规范）
- 类型
- 简介
- 故事梗概
- 文化意义
- 相关地点
- 关键词
- 来源（标注质量级别）
- 可信度与核实（合并评级+核实方式）
- 待核实点

**故事梗概长度规范**：
- 历史人物条目 ≤ 25行
- 名胜古迹条目 ≤ 30行
- 其他类型条目 ≤ 20行
- 超长内容拆分为多个条目或精简

## 类型可选值

神话传说、民间故事、非遗、节庆习俗、地方戏曲、饮食文化、传统工艺、历史人物、地方掌故、名胜古迹、宗教信仰、民俗活动。

**类型选择原则**：
- 历史人物条目只用于以人物为主角的条目（如"周敦颐"）
- **名胜古迹**用于以地点/建筑/遗址为主角的条目，收录其来源、历史事迹、名人与它的故事、神话传说、古今变迁（如岳麓书院→名胜古迹，南岳衡山→名胜古迹）
- 地方掌故用于不以特定名胜古迹为主角的地方性事件、掌故轶事（如通道转兵→地方掌故，湘南湘西抗战游击→地方掌故）
- 名胜古迹与地方掌故的区分：名胜古迹有明确的实体地点且条目以该地点为主线；地方掌故则侧重事件本身而非某个特定景点
- 非遗和传统工艺有重叠时，已列入国家级非遗名录的优先归**非遗**（如湘绣→非遗）
- 同一故事兼具历史和神话元素时，按**主要性质**归类（屈原投江→神话传说，因端午起源属神话叙事）

**名胜古迹收录内容要求**：
- 来源：该名胜古迹的建造/形成起源、命名由来
- 历史事迹：在该地发生的重要历史事件
- 名人与它的故事：与该地有深厚关联的名人故事（不在该地出生的名人也需收录，如韩愈谒衡岳庙）
- 神话传说：围绕该地的民间神话和传说
- 古今变迁：如有古今地名变化或建筑毁建历史需标注

## 来源质量分级与交叉认证

### 来源质量分级

每个来源标注质量级别：

| 级别 | 定义 | 示例 |
|------|------|------|
| **A级** | 正史/一手文献 | 《史记》《宋史》《清史稿》，人物本人著作（《岳阳楼记》原文），考古发掘报告 |
| **B级** | 官方认定 | 国家级非遗名录，全国重点文物保护单位名录，地方志，纪念馆官方资料 |
| **C级** | 二手学术 | 学术专著（陈来《宋明理学》），专业论文，权威百科（百度百科/维基百科） |
| **D级** | 民间口述 | 民间传说，地方口述传统，无书面佐证的故事 |

### 可信度等级与交叉认证要求

| 可信度 | 交叉认证要求 | 说明 |
|--------|-------------|------|
| **可靠** | ≥2个A级来源互相佐证 | 正史记载+考古实物；人物著作+正史传记 |
| **基本可靠** | ≥1个A级 + ≥1个B/C级来源 | 有正史或官方依据，另有佐证 |
| **待核实** | 仅C/D级来源，或单一A级来源无其他佐证 | 缺乏交叉佐证 |
| **存疑** | 仅D级来源且内容自相矛盾 | 民间口述且无其他佐证 |

### 交叉认证三条原则

1. **不同类型才算交叉**：两个同类型来源不算交叉佐证（两个C级百科词条不算；一个A级正史+一个D级民间传说算交叉——正史佐证了民间传说的核心情节）
2. **故事内部一致性**：同一故事的多个版本讲述同一核心情节 → 核心情节可信度高于版本细节差异
3. **混合标注**：条目内区分哪些内容有交叉佐证（基本可靠）、哪些缺乏佐证（待核实）

### 可信度与核实字段格式

合并原"可信度"+"核实方法"为一个字段：

`可信度与核实：基本可靠（A级《史记·屈原贾生列传》+ B级汨罗端午非遗名录交叉佐证；实物遗迹可实地考察）`

混合标注格式：

`可信度与核实：混合——正史记载部分基本可靠（A级《史记》+B级非遗名录），民间传说部分待核实（仅D级口述来源）`

### 来源字段格式

来源标注质量级别，重复来源使用省份文件来源索引编号：

```
### 来源
- [S2] 《屈原贾生列传》（A级）
- 汨罗江畔端午习俗国家级非遗项目资料（B级）
- [S3] ich.gov.cn（C级）
- 汨罗江畔民间口述传说（D级）
```

## 来源索引规则

每个省份文件头部设 `## 来源索引` 区域，首次出现的书籍/文献写完整信息并编号，后续条目只引用编号：

```
## 来源索引

[S1] 洪孟春主编《大地苍茫》，长沙晚报社编著，湖南人民出版社"文脉长沙"丛书
[S2] 《史记》，司马迁
[S3] 中国非物质文化遗产网 ich.gov.cn
[S4] 《宋史·道学传》
[S5] 黄宗羲《宋元学案·濂溪学案》
[S6] 湖南省博物馆官方资料
...
```

条目中来源字段写：`- [S1] 目录3：坐堂长沙` 代替重复完整出版社信息。新来源首次使用时在索引区添加编号，条目中写 `[SX] 章节名/页码`。

## 重要要求

不要编造来源。
无法确认的内容标记为"待核实"。
视频资料需要记录 BV 号、标题、UP 主、发布时间、链接。
同一故事存在多个地方版本时，要说明版本差异。

## 搜索流程优化

在搜集文化故事或人物传记时，遵循以下流程以减少 token 消耗：

1. **先查知识库**：使用 Grep/Glob 搜索知识库中是否已有相关条目
2. **有则补充**：如果已存在相关条目，直接编辑补充，不重新搜集
3. **无则外搜**：只有在知识库中确实没有相关内容时，才使用 WebSearch（2-3次精准搜索）
4. **外搜后录入**：搜索结果经审核后写入省份文件

**Token 消耗红线**：
- 已有条目补充 → WebSearch 2-3次 + 直接编辑（不启动 deep-research）
- 新条目录入 → WebSearch 2-3次 + 写入（仅大课题用 deep-research）
- deep-research 仅用于全新大课题（知识库完全空白 + 需要5+维度覆盖）
- 任职名人故事 → 已有独立人物条目时，名胜古迹/掌故条目只写简述+链接，不重复人物完整故事

视频内容录入流程：
1. 使用 `kb_fetch_video` 获取视频元数据
2. Claude Code 从视频中提取内容
3. 使用 `kb_ingest_video` 一键录入（获取元数据 + 创建来源 + 写入条目）
